# Portfolio - Gema Deprado
link al repositorio: https://github.com/gemadep/porfolioGema.git

Este repositorio contiene el código de mi portfolio web como diseñadora de producto. 
La página presenta mi enfoque de diseño basado en el equilibrio entre funcionalidad, estética y sostenibilidad, y reúne cuatro proyectos de diseño de producto desarrollados a partir de investigación, experimentación con materiales y reflexión conceptual.

## Proyectos

**AINA** - Bolso sostenible inspirado en la naturaleza y el doblado de la madera

**ECOTA** - Vajilla con enfoque funcional y consciente

**URA** - Sistema para fomentar el consumo responsable del agua mediante sensores

**T’UKU** - Botellero que combina principios físicos y trabajo artesanal

Cada proyecto incluye una descripción conceptual y del proceso de diseño.


## Sobre mí
La web incorpora una sección personal donde presento mi perfil como diseñadora de producto, mis intereses, experiencia y herramientas de trabajo, así como mi manera de entender el diseño: crear objetos con identidad, utilidad y significado.
